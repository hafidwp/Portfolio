Klasifikasi Jenis Beras Menggunakan CNN

Eksperimen penerapan Convolutional Neural Network (CNN) untuk mengklasifikasikan jenis beras berdasarkan bentuk bulir menggunakan dataset citra.

Klasifikasi Jenis Beras dengan CNN

Deskripsi

Proyek ini bertujuan untuk mengimplementasikan model Convolutional Neural Network (CNN) dalam mengklasifikasikan lima jenis beras berdasarkan bentuk bulirnya. Dataset diperoleh secara primer dan sekunder, lalu melalui tahap preprocessing sebelum dilatih dengan model CNN. Hasil pengujian menunjukkan akurasi mencapai 93% pada data uji.

Teknologi yang Digunakan

  • Python (Google Colab)
  • TensorFlow dan Keras
  • OpenCV untuk preprocessing citra
  • Matplotlib untuk visualisasi hasil
